

# Commit Guideline


Angular convention
```
<type>[optional scope]: <description>

[optional body]

[optional footer]
```

type:
- fix
- feat
- docs
- style
- refactor
- perf
- test
- BREAKING CHANGE


## Keywords to use 
Use imperative mood
- Main:
  - Add 
  - Remove
  - Update
  - Refactor
  - Fix
- Extra:
  - Change
  - Improve
  - Redo
  - Optimize
  - Revert
  - Correct
  - Release
